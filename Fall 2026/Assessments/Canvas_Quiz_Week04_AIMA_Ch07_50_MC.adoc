== Learning Targets

Students should be able to:

* Explain the role of a knowledge base in a knowledge-based agent.
* Distinguish TELL, ASK, axioms, inference, and knowledge representation languages.
* Explain how the Wumpus World illustrates reasoning under partial observability.
* Define syntax, semantics, models, satisfaction, entailment, soundness, and completeness.
* Interpret basic propositional logic connectives and truth conditions.
* Explain truth-table model checking and its computational cost.
* Distinguish model checking from theorem proving.
* Explain validity, satisfiability, proof by refutation, and conjunctive normal form.
* Describe resolution, definite clauses, Horn clauses, forward chaining, and backward chaining.
* Compare DPLL and WalkSAT as SAT-solving methods.
* Explain successor-state axioms, the frame problem, logical state estimation, hybrid agents, and SATPLAN.
