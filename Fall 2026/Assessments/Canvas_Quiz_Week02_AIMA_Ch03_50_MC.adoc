== Learning Targets

Students should be able to:

* Explain what a problem-solving agent does.
* Distinguish goal formulation, problem formulation, search, and execution.
* Define the parts of a search problem: initial state, actions, transition model, goal test, and action cost.
* Distinguish a state-space graph from a search tree.
* Explain nodes, frontiers, reached states, repeated states, cycles, and redundant paths.
* Evaluate search algorithms using completeness, cost optimality, time complexity, and space complexity.
* Compare breadth-first search, uniform-cost search, depth-first search, depth-limited search, iterative deepening search, and bidirectional search.
* Explain how heuristic functions guide informed search.
* Compare greedy best-first search, A-star search, weighted A-star search, IDA-star, RBFS, and SMA-star.
* Explain admissibility, consistency, dominance, relaxed problems, pattern databases, landmarks, and learned heuristics.
