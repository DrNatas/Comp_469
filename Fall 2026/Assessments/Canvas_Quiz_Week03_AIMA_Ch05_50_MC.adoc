== Learning Targets

Students should be able to:

* Explain why adversarial environments require different search methods from single-agent search.
* Define deterministic, two-player, turn-taking, perfect-information, zero-sum games.
* Identify the formal components of a game: initial state, player to move, legal actions, transition model, terminal test, and utility function.
* Explain game trees, terminal states, utilities, ply, and strategies.
* Compute and interpret minimax values in two-player zero-sum games.
* Explain why minimax assumes optimal play by both players.
* Describe how minimax generalizes to multiplayer games with utility vectors.
* Explain alpha-beta pruning and why it can return the same decision as minimax while evaluating fewer nodes.
* Explain how move ordering, killer moves, and transposition tables improve game-tree search.
* Explain heuristic alpha-beta search, evaluation functions, cutoff tests, quiescence search, horizon effects, and forward pruning.
