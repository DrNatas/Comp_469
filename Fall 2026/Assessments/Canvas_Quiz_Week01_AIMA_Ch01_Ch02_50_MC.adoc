== Suggested Canvas Setup

Course:: COMP 469 - Artificial Intelligence/Neural Nets
Week:: Week 1
Reading:: AIMA Chapter 1 - Introduction and Chapter 2 - Intelligent Agents
Lab Alignment:: Lab 01 - Vacuum World Agents
Recommended Use:: Canvas quiz, reading check, lab readiness quiz, or midterm review
Question Style:: Multiple choice
Recommended Points:: 1 point each
Recommended Time Limit:: 35-50 minutes for all 50 questions, or 15-25 minutes for a random draw of 20-25 questions
Canvas Notes:: Enter each item as a Multiple Choice question. Use the listed Correct Answer letter as the correct option. Randomize question order and answer-choice order if using the full bank.

== Learning Targets

Students should be able to:

* Explain the four major AIMA approaches to defining AI.
* Distinguish human-centered AI from rationality-centered AI.
* Explain the Turing test, total Turing test, and rational-agent approach.
* Identify major AI capabilities, foundations, benefits, and risks.
* Define agents, percepts, percept sequences, agent functions, and agent programs.
* Explain rationality in terms of expected performance.
* Apply PEAS to the Vacuum World lab.
* Classify task environments using AIMA Chapter 2 vocabulary.
* Compare table-driven, reflex, model-based, goal-based, utility-based, and learning agents.
