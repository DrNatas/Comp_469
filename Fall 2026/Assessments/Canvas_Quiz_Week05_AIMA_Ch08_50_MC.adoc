== Learning Targets

Students should be able to:

* Explain why first-order logic is more expressive than propositional logic.
* Distinguish declarative, procedural, compositional, ambiguous, and context-dependent representations.
* Explain the ontological commitment of first-order logic to objects and relations.
* Define domains, objects, interpretations, constant symbols, predicate symbols, function symbols, and arity.
* Explain terms, complex terms, ground terms, atomic sentences, and complex sentences.
* Interpret universal and existential quantifiers.
* Recognize common quantifier mistakes, including using conjunction with universal quantification or implication with existential quantification in the wrong place.
* Explain nested quantifiers and why quantifier order matters.
* Explain the connections between universal and existential quantification through negation.
* Compare standard first-order semantics with database semantics.
* Describe the knowledge-engineering process for building a first-order knowledge base.
* Apply first-order logic ideas to the electronic circuits domain.
