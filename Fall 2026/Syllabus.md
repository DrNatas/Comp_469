# COMP 469: Introduction to Artificial Intelligence
**CSU Channel Islands | Spring 2026**

This document serves as the syllabus for COMP 469, an upper-division survey course designed to provide a hands-on, rigorous exploration of the techniques used to make computers appear intelligent.

---

### Course Identity & Goal
**Course:** COMP 469 - Introduction to Artificial Intelligence
**Units:** 3 (2 Lecture Hours / 3 Lab Hours per week)
**Prerequisites:** COMP 350 and COMP 362 (Strong foundation in programming/discrete structures required)
**Format:** Synchronous, In-person (Hybrid elements expected for online resource use).

**Course Description:**
A hands-on exploration of the use of computers to perform computations normally associated with intelligence, pattern formation and recognition using a variety of symbolic and sub-symbolic methods. Knowledge acquisition, representation, and maintenance will be covered. We will progress from classical problem-solving techniques to modern machine learning paradigms.

**Goal:** Students will leave this course able to explain core AI concepts, implement small AI systems in code, and critically evaluate the tradeoffs (interpretability vs. performance, data needs, etc.) of different AI methodologies.

### Student Learning Outcomes (SLOs)
On completion of the course, students should be able to:
1.  **Model Intelligence:** Formally model problems as state-space graphs and select appropriate search algorithms (e.g., BFS, A*).
2.  **Reason Symbolically:** Represent knowledge using formal logic (First-Order Logic) and implement basic inference mechanisms.
3.  **Handle Uncertainty:** Apply probabilistic methods, including Bayes' Theorem, to reason with incomplete or uncertain information.
4.  **Implement Machine Learning:** Build and train fundamental machine learning models (e.g., Perceptrons, Linear Regression) using common ML libraries.
5.  **Critique AI Systems:** Articulate the core trade-offs, failure modes, and ethical implications of both symbolic and sub-symbolic AI approaches.

### Tentative Course Schedule (15 Weeks)

| Week | Dates (Example) | Core Concept | Key Topics & Focus | Lab/Assignment Focus |
| :--- | :--- | :--- | :--- | :--- |
| **1** | Sept 2-5 | **AI Agents & Foundations** | Defining intelligence; Rational Agents; PEAS framework. | Lab: Simple Grid World Simulation. |
| **2** | Sept 9-12 | **State-Space Search** | Problem formulation; BFS, DFS, UCS; Graph representation. | Lab: Implementing foundational Graph Search algorithms. |
| **3** | Sept 16-19 | **Informed Search & Planning** | Heuristics (h(n)); Admissibility; A* Search; Simple Planning (STRIPS). | Lab: Solving classic pathfinding problems (e.g., 8-Puzzle). |
| **4** | Sept 23-26 | **Knowledge Representation & Logic** | Propositional Logic vs. First-Order Logic; Knowledge Bases. | Lab: Implementing basic logical inference rules. |
| **5** | Sept 30-Oct 3 | **Advanced Reasoning** | Inference (Resolution, Forward/Backward Chaining); Expert Systems overview. | Lab: Building a simple, rule-based reasoning engine. |
| **6** | Oct 7-10 | **Probability & Uncertainty I** | Basic Probability Theory; Conditional Probability; Bayes’ Theorem derivations. | Lab: Computational exercises verifying Bayes' Theorem. |
| **7** | Oct 14-17 | **Probability & Uncertainty II** | Bayesian Networks; Graphical Models; Causal Inference (Overview). | Lab: Building and querying a small Bayesian Network. |
| **8** | Oct 21-24 | **MIDTERM REVIEW & PROJECT CHECKPOINT** | *Review of Symbolic AI, Search, and Logic.* | **Midterm Exam.** Mini-Project Checkpoint/Debugging Session. |
| **9** | Oct 28-31 | **Sub-Symbolic I: Machine Learning Intro** | Introduction to ML; The Perceptron; Linear Separability. | Lab: Implementing a basic Perceptron classifier. |
| **10** | Nov 4-7 | **Machine Learning II: Core Models** | Deepening NN understanding; Gradient Descent; Backpropagation mechanics. | Lab: Training a simple NN (e.g., XOR problem). |
| **11** | Nov 11-14 | **ML Paradigms & Optimization** | Supervised vs. Unsupervised Learning; Clustering (K-Means); Bias/Variance Tradeoff. | Lab: Applying K-Means or implementing Linear Regression. |
| **12** | Nov 18-21 | **Reinforcement Learning (RL)** | MDPs; Value Function; Policy Iteration & Value Iteration. | Lab: Solving a simple Grid World using Value Iteration. |
| **13** | Nov 25-28 | **Advanced Topics / Flex** | *Instructor choice:* Deep Learning Architecture Overview OR Advanced Planning (e.g., PDDL). | Lab: Exploration/Advanced Topic Implementation. |
| **14** | Dec 2-5 | **Ethics, Limitations, & Future AI** | Bias in Data, Fairness, Interpretability (XAI), and Responsible AI guidelines. | Case Study Analysis: Evaluating real-world AI failures. |
| **15** | Dec 9-12 | **Final Synthesis** | Review of the entire pipeline: From logic statements to deployed ML models. | Final Project Presentations / Comprehensive Review. |
