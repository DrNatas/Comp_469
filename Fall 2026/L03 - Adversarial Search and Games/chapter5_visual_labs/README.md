# Chapter 5 Visual Labs: Adversarial Search and Games

Five standalone Jupyter notebooks for university-level instruction. Each notebook includes chapter notes, complete local Python code, visual search displays, and student practice cells.

## Files
1. Lab_05A_Minimax_TicTacToe_Game_Trees.ipynb
2. Lab_05B_Alpha_Beta_Pruning_Move_Ordering.ipynb
3. Lab_05C_Heuristic_Alpha_Beta_Connect4.ipynb
4. Lab_05D_MCTS_Stochastic_Games.ipynb
5. Lab_05E_Partially_Observable_Games_Limitations.ipynb

## Requirements
Python 3 with Jupyter, matplotlib, and optionally ipywidgets for sliders/buttons.

```bash
pip install notebook matplotlib ipywidgets
```

The notebooks still show static first/last frames if ipywidgets is unavailable.

## Environment Activation
```bash
source .env/bin/activate
```
